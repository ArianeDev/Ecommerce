# App biblioteca
O aplicativo consiste em um aplicativo desenvolvido em Flutter que simula um aplicativo de ecommerce. 
Possível ser acessado por dois tipo de usuário, sendo eles o usuário comum e o admin.

## Funcionalidades
- Login

###Usuário comum
- Lista de livros
- Carrinho de compra
- Finalização da compra

###Admin
- Cadastro de novos livros
- Atualização dos livros
- Exclusão dos livros
- Exibição dos Livros

## Tecnologias utilizadas
- Flutter
- Dart
- Firebase

## Como executar o projeto

```
  # Clonar o repositório
  git clone

  # Acessar a pasta
  cd Front

  # Instalar dependências
  flutter pub get

  # Executar no emulador ou dispositivo físico
  flutter run
```
